- 사전체크 - 오늘의 단어-
```
0 : 확실히 아는 단어
1 : 복습하고 싶은 단어
2 : 잊은 단어
3 : 처음보는 단어
```

- 시간 - 복습이 필요한 단어 나누기
```
0 : 바로해결 : 10초 안
1 : 긴가 민가 : 10~20초
2 : 모름 : 20초 이상
```
- 정답
```
0 : 정답
1 : 모름
2 : 오답
```

